# Simple Shift Cipher Tool

A lightweight, beginner-friendly Python script that allows you to encrypt and decrypt text using a custom shift code.

## 📖 What is this?
This project is a implementation of a **Shift Cipher**. It works by taking each character in your text and "shifting" its value in the Unicode table by a number you provide. 

For example, if you enter a shift code of `5`, the letter `A` becomes `F`. To get your original message back, you simply use the same code to decrypt it.
